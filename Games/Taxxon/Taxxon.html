About:

TaXXon was very obviously set out to by a Zaxxon (Sega, 1982) inspired game. axonometric perspective, which gave original Zaxxon it's name is achieved using WebGL, supported by my 3D game engine (called ENGINE) written in JS and GLSL.

I played a c64 port of Zaxxon and I specifically dislike a space part of it, so my version ignores that silly part.
